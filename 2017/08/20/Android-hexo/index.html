    • 工具

    Neoterm(酷安网可以下载)

    之前用过Termux 但是因为系统的原因tsu也没有权限使用hexo而su又不能执行hexo的命令折腾一番找到了可以用su正常执行hexo的软件Neoterm

    /data/data/io.neoterm/files/ 这个是Linux的根目录

    环境配置

    ` bash
    apt update && apt upgrade //更新apt
    apt install coreutils //安装一些基本工具
    apt install git //安装git
    apt install nodejs //安装node.js

    `

    安装HEXO

    ` bash
    npm install hexo-cli -g //安装hexo
    mkdir hexo //创建博客文件夹
    cd hexo //进入博客文件夹
    hexo init //初始化博客

    `

    部署至GitHub

    注册账号 创建epository这里就不写了网上教程有很多
    编辑本地Hexo目录下文件_comfig.yml，在最后添加如下代码（在你修改时，把 gitzhaoyang 要替换成你自己的用户名）
    ` bash
    deploy: type: git repository: http://github.com/Gitzhaoyang/gitzhaoyang.github.io.git branch: master

    `

    执行以下指令即可完成部署
    ` bash
    $ hexo clean
    $ hexo generate
    $ hexo deploy //输入GitHub账户和密码即可
    `
