    环境配置

    • 首先更新下apt
      apt update && apt upgrade
      –安装一些基本工具
      apt install coreutils
      –安装git
      apt install git
      安装node.js
      apt install nodejs

      安装hexo

      –安装hexo
      npm install hexo-cli -g
      然后在home目录下新建一个hexo文件夹
    • 进入hexo目录
      cd hexo(报错的话输入su回车再进入hexo)
    • 初始化hexo
      hexo init
      之后输入hexo clean
      hexo g
      hexo s
      访问127.0.0.1:4000就可以看的效果了
